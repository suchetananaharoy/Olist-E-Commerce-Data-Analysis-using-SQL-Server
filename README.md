# Olist Brazilian E-Commerce Data Analysis

## 📌 Overview
This repository contains the **Olist Brazilian E-Commerce Public Dataset** prepared for **SQL Server Management Studio (SSMS)**.  
It includes cleaned tables, SQL scripts, and 50 practice queries for **data analysis, aggregation, joins, and reporting**.  
Designed for **data analyst interview preparation** and portfolio projects.

---

## 📂 Dataset
The dataset contains real e-commerce transactions in Brazil and is divided into multiple tables:

| Table | Description |
|-------|-------------|
| `customers` | Customer information (city, state, unique ID) |
| `orders` | Orders with timestamps and status |
| `order_items` | Items in each order with product and seller info |
| `products` | Product information including category |
| `sellers` | Seller information |
| `payments` | Payment type and values per order |
| `reviews` | Customer review scores for orders |
| `category_translation` | Product category translations |

**Source:** [Kaggle - Olist E-Commerce Dataset](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce)
